projects i've done:

Music Mentor

This online tool helps users learn music theory in a fun and innovative way. The design of the web application makes learn the core concepts of music theory effortless. Users can experiment with the various functions the web app supports including converting between music scales and creating chords. Furthermore, users can use the companion Messenger bot which augments the learning experience.

This is a personal project started during the winter break of '16-17 and finished in February '17.
Created with: JavaScript, Node.js, Python, Flask, HTML, CSS

College Care

A web application created as part of a NY State Making College Possible Coding competition. The purpose of our application is to provide valuable resources, mentoring opportunities, and an open forum to aid students to find an affordable college education in New York State. We were placed top 9 in whole competition out of over 60 competitors.

This was a 2-person project done in February '17.
Team members: Shaan Sheikh
Created with: HTML, CSS, Bootstrap, Python, Flask, SQLite3


Zero Waste

A website created along the theme of helping the environment. Our site aims to educate and help consumers of electronics in NYC to reduce their electronic wastes to ideally "zero".

This was a hackathon project submitted to Social Hack Day in Janurary '17.
Team Members: Jon Reyes, Kitty Liu
Created with: HTML, CSS, JavaScript, d3.js

Exercise Counter

We wanted to use LittleBits to create a device that we may use in everyday life. We came up with the exercise counter, which uses an Arduino bit and some sensors to count the number of repetitions for a number of exercises, such as pushups and squats.

This was a hackathon project submitted to HACKfEAT in October '16.
Team Members: Zhengyang Liu
Created with: Arduino, LittleBits
